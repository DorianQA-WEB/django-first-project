# Django First Project

Простой веб-сайт на Django, демонстрирующий базовые возможности фреймворка: маршрутизацию, представления, шаблоны, работу с данными и тестирование. Проект может использоваться как учебный пример или стартовая точка для разработки.

## 📌 Описание функционала

Проект включает следующие разделы:

- Главная страница (/) — приветствие и краткое описание.
- О сайте (/about/) — информация о проекте и авторе.
- Контакты (/contact/) — данные для связи.
- Профиль пользователя (/profile/) — отображение данных пользователя.
- Поиск книг (/books/search/) — страница поиска (пример расширяемости).
- Пользователи и данные — динамические URL с параметрами (например, /user/Tom/38/).

Также реализованы:
- Навигационное меню.
- Базовый шаблон с наследованием.
- Обработка GET-параметров.
- Юнит-тесты для проверки доступности страниц.

## 🛠 Технологии

- Python — язык программирования.
- Django — веб-фреймворк.
- SQLite — база данных (по умолчанию).
- HTML / CSS — верстка и оформление.
- unittest — тестирование.

## 🚀 Запуск проекта

1. Клонируйте репозиторий:
git clone https://github.com/ваш-ник/Django_FirstProject.git
cd Django_FirstProject

2. Создайте виртуальное окружение:
python -m venv .venv

Активация окружения:
- Windows:
.venv\Scripts\activate
- Linux / macOS:
source .venv/bin/activate

3. Установите зависимости:
pip install django

4. Запустите сервер:
python manage.py runserver

Перейдите в браузер: http://127.0.0.1:8000

## 🧪 Запуск тестов

Запустите тесты командой:
python manage.py test
